# FastAPI starter kit

[FastAPI](https://fastapi.tiangolo.com/) simple starter kit

## Rewuirement

Python 3.8+

## Installation

Use [poetry](https://python-poetry.org/docs/)

```bash
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $ poetry install
```

## Usage

### Running

```bash
(venv) $ python app.py
```

### Testing

```bash
(venv) $ pytest .
```

### Docker

```bash
docker-compose -f docker-compose-local.yml up
```
